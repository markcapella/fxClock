## Description

        A minimally sized Java FX app, digital clock for
    Linux PLASMA & GNOME desktops, with a beeping alarm.


## Installation

###     CD into source repo.

    make
    make run

    sudo make install
    sudo make uninstall

    make clean


## Usage

###     GUI Desktop:

        Click the fxClock desktop icon that's added to your systems menu
    under "Utilities".


###     Command Line provides:

    startFxClock
    stopFxClock
