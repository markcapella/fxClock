## Description

###    A minimally sized java FX app, digital clock for Linux Plasma, with a beeping alarm.


## Installation

###    CD into source repo.

    make
    make run
    make install
    make uninstall
    make clean


## Usage

    Double click the icon that's added to your desktop.
