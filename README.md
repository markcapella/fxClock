## Description

###       A minimally sized Java FX app, digital clock for Linux PLASMA & GNOME desktops,
###    with a beeping alarm.


## Installation

###    CD into source repo.

    make
    make run

    sudo make install
    sudo make uninstall

    make clean


## Usage

    Find and click the icon that's added to your systems menu.
